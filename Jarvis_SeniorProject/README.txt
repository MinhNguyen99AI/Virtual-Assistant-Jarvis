Instruction on using Jarvis:

1. RUN:

pip install -r requirements.txt

in terminals to install required libraries

2. Run main.py

3. Enjoy.

Start with 'Hello Jarvis'

Then give command by saying:
"Jarvis":
- send email
- play [name of the song]
- open [name of application]
- close [name of application]
- search [something]
- check weather today
- check weather today in [location]
- check rating [movies name]
- look up wikipedia for [something]
- define [word]
- set alarm at [time]
